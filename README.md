# Doubly Linked List in C

Implementation of a doubly linked list with:

- Insert at beginning
- Insert at end
- Insert after a key
- Delete from beginning
- Delete from end
- Delete by value
- Forward traversal
- Backward traversal

Concepts practiced:
- Dynamic memory allocation
- Structures
- Pointers
- Prev/Next node manipulation
